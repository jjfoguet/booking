Exploring Geographic Insights

Interactive Map of North Argentina

In addition to other spatial tourism databases ( OpenStreetMap and Google Maps APIs), I extracted, from a popular web platform, the coordinates and reviews of hotels in Northwest Argentina. These reviews provide valuable insights: in one hand they reveal the nationality of the reviewers, which helps in creating flow maps; on the other hand, their quantity highlights the level of interest in specific geographic locations.

The extraction process involves two steps. First, I query a 'hidden' endpoint that powers the webmap associated with each hotel. This requires providing the coordinates of a bounding box of interest as parameters. From this request, I retrieve the establishment ID, the number of reviews, and other details that are then used to make a second request to the 'hidden' reviews API for each hotel. You can check out the code here.

The map above is dynamically rendered using the Folium library. The data represents aggregated observations and highlights key trends in visitor demographics.
